# TETO 1.0.1 维护清单

# 一、当前阶段目标

只做 1.0 发布后小修补，不扩展 1.1，不新增大模块。

## 二、P0：影响使用体验的问题

- Stats 无法展示各任务趋势
- Projects 未展示预计完成时间
- Daily Record / Diary 展示方式导致大量空白未填写区域
- 页面布局无法折叠，信息密度过高

## 三、P1：明显想优化但暂不立刻做

- Daily Record 增加表格视图
- 卡片支持折叠
- 某些区域支持尺寸调整
- Dashboard / Stats 信息展示更紧凑

## 四、P2：先记录，后续版本再决定

- 任务 / 状态 / 进度 DIY
- 目标值系统
- 基于目标值的完成度
- 基于当前速度的任务完成预测
- 自由拖拽布局

## 五、当前不做

- 第二大脑
- 财务
- AI 自动解析
- 复杂动态配置系统
- 多人协作
